# 后续迭代任务

> 以下任务按优先级排列。完成顺序可调整。

## P0 - 核心功能完善

- [ ] **AI 类型判断**：接入 LLM API，根据想法内容自动判断类型和危险等级，用户只需输入内容。
- [ ] **AI 系统建议**：根据想法内容和上下文，由 AI 生成个性化建议，替代静态文案。
- [ ] **数据导出**：支持将想法列表导出为 JSON / Markdown / CSV。
- [ ] **多设备同步**：接入后端服务，支持多设备间同步想法。

## P1 - 体验增强

- [ ] **定时提醒**：到达释放日期时，通过浏览器通知提醒用户。
- [ ] **统计看板**：展示收容趋势、类型分布、释放率等统计图表。
- [ ] **批量操作**：支持多选想法进行批量释放、归档等。
- [ ] **撤销操作**：误操作后可撤销（已释放 → 可释放等）。
- [ ] **暗色/亮色主题切换**：可选。

## P2 - 扩展功能

- [ ] **标签系统**：除了类型外，支持自定义标签。
- [ ] **想法关联**：在合并时，记录关联关系，后续可查看完整历史。
- [ ] **冷却期自定义**：允许用户手动调整冷却天数。
- [ ] **想法分类树**：将想法组织成树形结构，方便回顾。
- [ ] **导入功能**：支持从其他工具（如备忘录、Notion）导入想法。

## P3 - 后端 & 基础设施

- [ ] **REST API**：使用 Hono + Cloudflare Workers / Node.js 构建后端 API。
- [ ] **数据库**：使用 SQLite / D1 / PostgreSQL 替代 localStorage。
- [ ] **用户系统**：支持账号注册、登录。
- [ ] **自动化**：定时任务检查冷却状态，发送推送通知。

## P4 - AI 深度集成

- [ ] **相似想法检测**：录入时检测是否已有类似想法，提示合并或关联。
- [ ] **冷却期预测**：根据历史行为，预测最合适的冷却时间。
- [ ] **每周回顾报告**：AI 自动生成每周想法回顾报告。
- [ ] **对话式回顾**：以对话方式与 AI 一起回顾和沉淀想法。
