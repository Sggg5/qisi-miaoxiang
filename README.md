# 奇思妙想（Qisi Miaoxiang）

> 先把想法收起来，等它冷却后再决定。

**奇思妙想**是一个帮助用户暂存、冷却和整理突然冒出来的想法的小工具。它不是 Todo，也不是项目管理工具，而是用来防止新想法打断当前任务。

你可以快速记录一个想法，系统会根据类型和危险等级自动设置冷却时间。到期后，你再决定是否释放、归档、继续观察，或合并到已有项目。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- **Vue 3**（Composition API）
- **Vite**
- **Pinia**（状态管理）
- **Vue Router**
- 数据持久化：localStorage

## 功能特性

- 快速收容想法，包含内容、类型、危险等级等字段
- 8 种想法类型覆盖常见「冲动」场景
- 1～5 级危险等级，自动设置 1～14 天冷却期
- 5 种状态流转：观察中 → 可释放 → 已释放 / 已归档 / 已合并
- 到达释放日期后自动标记为可释放
- 可释放想法支持：释放成项目、继续观察（+3 天）、归档为废案、合并到旧项目
- 按类型、状态、危险等级筛选
- 关键词搜索
- 桌面端 + 移动端适配
